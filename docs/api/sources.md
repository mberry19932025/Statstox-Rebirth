# StatStox API Sources (Free / Free-Tier)

These sources are suitable for prototyping and staging. Always review each provider's Terms of Service, rate limits, and attribution requirements before shipping.

## Sports Data APIs (Top 5)
1) TheSportsDB — multi-sport JSON API (free)
2) API-Sports — multi-sport coverage with free tier limits
3) Sportmonks — strong football/soccer data (trial/free tier)
4) Fantasy Premier League (FPL) API — official EPL fantasy data (public endpoints)
5) MLB Stats API — unofficial but widely used endpoints for MLB stats/schedules

## API Building & Testing Tools (Top 5)
1) Thunder Client — VS Code REST client for quick testing
2) Postman — free tier for request collections and docs
3) Swagger / OpenAPI — open standard for API specs + UI docs
4) Insomnia — lightweight API client (free tier)
5) Mockoon — local mock server for frontend testing

## Optional Market Data APIs (Top 5)
If you want a real-market educational layer, these free tiers are common:
1) Alpha Vantage
2) Finnhub
3) IEX Cloud (free tier)
4) Twelve Data
5) Marketstack

## Compliance Notes
- Do not scrape sites that disallow it; use official APIs only.
- Respect provider rate limits and caching guidance.
- Display attribution if required by the provider.
- Keep DFS logic and Market logic separate to avoid policy conflicts.
- Never store or expose API keys in the frontend.
